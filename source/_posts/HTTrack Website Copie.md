**HTTrack 是一款既免费又实用的工具，它能创建和目标网站一样的离线脱机副本。内容包括原始网站的所有网页、链接、图片等。作用是为了减少与目标系统交互,避免触发目标系统的报警机制。**

<!-- more -->

## HTTrack 安装

```
apt-get install webhttrack
```
`Kali Linux` 自己带。


## HTTrack 使用

```
httrack
```
